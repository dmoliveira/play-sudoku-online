# Sudoku Sakura

Japanese-inspired Sudoku for GitHub Pages, built with plain HTML, CSS, and JavaScript.

## Features

- Easy, medium, hard, and expert puzzles
- Timer and best-time tracking with `localStorage`
- Notes mode
- Optional wrong-guess highlighting
- Responsive board-first layout
- SEO-friendly landing content
- Top menu with Diego Marinho links

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Set the source to deploy from the main branch root.
4. Update canonical URLs in `index.html`, `robots.txt`, and `sitemap.xml` if the repo name changes.
